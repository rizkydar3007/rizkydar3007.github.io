Tugas Rangkuman GIT

1.GIT

Git adalah sistem pengontrol versi (Version Control System) pada proyek perangkat lunak yang diciptakan oleh Linus Torvalds.

2.GitLab

Gitlab adalah layanan yang menyediakan akses remote ke Git repositories. layanan ini menyediakan fitur tambahan yang dirancang untuk membantu mengelola siklus pengembangan software dan mempermudah dalam share data .

3.GitBash

Git Bash adalah aplikasi untuk lingkungan Microsoft Windows yang menyediakan lapisan emulasi untuk pengalaman Git command line,untuk membantu Control atau remote suatu projek.

*Git Init

Untuk Mengubah Folder biasa menjadi Folder repository sehingga disebut Repo karna sudah ada name master pada teminal

*Git Config

untuk melihat semua konfigurasi yang ada digit dan samakan user.name dan user.email seperti gitlab

*Git Status

Untuk memberitahu file yang baru atau berubah yang belum kita commit

*Git Add

untuk menambahkan beberapa file ke tracing area

*Git Commit

untuk mencatat perubahan yang dilakukan

*Git Remote

untuk berinteraksi dengan server/gitlab

*Git Push

untuk mengirim commit ke server/gitlab

*Git Log

untuk melihat semua histori commit yang dilakukan

*Git checkout

untuk menganti ke branch lain

*Git Branch

untuk membuat branch baru

*Git Branch -d

untuk menghapus branch yang sudah dibuat

4.SouceTree

SouceTree adalah layanan GUI untuk mengakses GIT sama seperti GITbash hanya saja lebih muda mengunakan SouceTree karna cukup klik2 saja

*Add File

Memasukan File yang akan kita pakai

1.Garis merah menandakan teks yang belum diubah,sedangkan baris teks yang sudah diperbarui

2.ialah file yang sudah kita perbarui melalui vscode atau yang lain

4.untuk mengatasi suatu Konflik dibutuhkan branch yang baru sehingga memudahkan dalam menyelesaikan masalah tanpa mengubah file pada ke2 branch dan branch konflik harus mengikuti isi dari branch master

5.commit mencatat perubahan yang dilakukan(file index.html)

6.Mengirimkan commit ke server sama seperti gitbash
